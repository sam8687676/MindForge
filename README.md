# MindForge

MindForge is an AI-powered personal productivity and home assistant designed for students, professionals, and makers.

## Features

* Desktop GUI built with Python
* Voice interaction
* User memory system
* Conversation logging
* MQTT integration for ESP32 devices
* Offline-first architecture

## Planned Features

* Task management
* Calendar integration
* Smart reminders
* Wake-word activation
* Study planning assistant
* Home automation
* Personal knowledge memory
* IoT dashboard

## Technology Stack

* Python
* Tkinter
* MQTT
* JSON
* pyttsx3
* Transformers
